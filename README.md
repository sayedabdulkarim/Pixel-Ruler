# Pixel Ruler

A Chrome extension to measure any element on a webpage with precision.

## Features

- **Measure Mode** - Drag to measure any area on the page
- **Element Mode** - Hover to highlight and measure DOM elements
- **Guides Mode** - Add horizontal/vertical guide lines
- **Multiple Units** - px, rem, %, vw, vh
- **Grid Overlay** - Optional grid for alignment
- **Custom Colors** - Choose your ruler color

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `ESC` | Deactivate/Cancel |
| `C` | Clear measurements |
| `G` | Toggle grid |
| `1-5` | Switch units |

## Installation

1. Clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder

## Usage

1. Click the extension icon
2. Select a mode (Measure/Element/Guides)
3. Choose your preferred unit
4. Click "Activate Ruler"
5. Start measuring!
